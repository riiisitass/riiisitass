## 👨‍💻 Sobre mí

Soy **ranitax** (alias), un desarrollador autodidacta con una fuerte pasión por la **ciberseguridad** y el **análisis de redes**. Mi enfoque principal es el desarrollo de herramientas en Python para auditoría de dispositivos IoT y detección de vulnerabilidades en infraestructuras críticas.

🔹 **Intereses:**  
- Seguridad ofensiva y defensiva  
- Escaneo de redes y descubrimiento de dispositivos  
- Automatización con Python  
- Análisis de tráfico de red  

🔹 **Estado actual:**  
Desarrollando un escáner de cámaras IP para Latinoamérica y España (proyecto privado en fase avanzada).
